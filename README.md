## Description

This project analyzes NYC Taxi trip data, calculating monthly average trip lengths and 45-day rolling averages.

## Setup

### Requirements

- Python 3.8+
- pandas
- matplotlib
- seaborn
- pyarrow
- pyspark

### Installation

Install the required packages:

```bash
pip install -r requirements.txt
```
